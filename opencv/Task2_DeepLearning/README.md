# Task 1 Deep Learning
## 学习内容
 - python 版 pytorch库的安装与使用
 - 创建神经网络模型类
 - 多层感知机
 - 激活函数
 - 卷积神经网络
 - 池化层
 - 损失函数
 - 优化器
 - yolov11的训练与使用
## 任务要求
 - 使用多层感知机实现XOR(异或)，输入两个布尔值(0 or 1)输出两个布尔值异或运算后的结果 20%
 - 使用卷积神经网络识别自己书写的汉字（单个汉字识别），要求能识别不少于二十个汉字 30%
 - 使用yolov11识别至少9种垃圾（可回收，有害垃圾，厨余垃圾至少各三种）40%
## 提交说明
 - 所有数据集及模型结构均自拟
